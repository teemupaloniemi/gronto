Basics

This form is used to collect course information for THESIS-X. If you are interested on why you can read some ideas from github.

How to use this form?

Fill in the form for all courses you are responsible for. If multiple people are responsible for the course we recommend working together.

For basic course information — i.e. name, code and credits — filling the input fields is enough.

For list formatted data — i.e. prerequisites and outcomes — you must select each item either by clicking them or using the Ctrl-N shortcut (for 1 ≤ N ≤ 8). For prerequisites and outcomes you are restricted to use only the ACM Computing Classification System ontology on the right. Most guidelines on usage from this document apply but we do not use weighting (yet).

When you feel like you have provided all necessary information for the form send an email to me containing the data. The form provides a link at the bottom that prefills the email for you.
